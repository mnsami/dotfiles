# TypeScript Domain

## Purpose
TypeScript development standards for building type-safe React and Node.js applications.

## Scope
- React frontend applications
- Node.js backend services
- Next.js full-stack applications
- Type-safe API integrations

## Responsibilities
- Enforce TypeScript best practices and strict typing
- Apply React patterns (hooks, composition, state management)
- Guide component architecture using Pragmatic Atomic Design
- Ensure proper error handling and validation

## Interaction Points
- Works with ui-ux-expert for component design
- Works with testing-expert for React Testing Library patterns
- Works with architect for frontend architecture decisions
